# 📖 Chat2Ebook：所见即所得

一个出于个人需求写出来的鸡肋小插件，顾名思义，主要功能是把聊天对话导出为包含渲染的电子书，即在酒馆页面看到是什么样式，导出的电子书就是什么样式。

做这个的原因是个人喜欢偏向小说式的推进剧情，并且很喜欢开一些HTML小装饰，但我能找到的导出插件基本都是纯文本格式的，所以用Gemini大人搓了一个可以导出渲染电子书的鸡肋小插件。

各种意义上的新人，学艺不精，有bug的话我给大家跪一个（？）

---

**目前支持导出格式：**

- 📱 **EPUB**
- 📘 **Word (.doc)**
- 🌐 **HTML**
- 📄 **TXT**（纯文本，无渲染）

## 📦 安装

1.  打开酒馆。
2.  进入 **Extensions (扩展)** 菜单。
3.  点击 **Install Extension (安装扩展)**。
4.  在 URL 栏中粘贴本仓库地址：
    ```
    https://github.com/sixwater6h2o/Chat2Ebook
    ```
    或
    ```
    https://gitee.com/sixwater_admin/Chat2Ebook
    ```
6.  点击 **Install**。安装完成后刷新页面即可。

## 📖 使用指南

1.  在酒馆顶部的扩展栏中找到 **Chat2Ebook**。
2.  点击标题 **📖 Chat2Ebook** 展开面板。
3.  **配置选项**：
    * **书名/作者**：默认自动抓取，也可手动修改。
    * **楼层范围**：选择要导出的起始和结束楼层。
    * **EPUB分章**：多少楼层视为一章。
    * **导出选项**：勾选是否包含用户/AI，或隐藏 AI 名字。
4.  **重要提示**：
    * 由于抓取逻辑的问题，**没有加载的聊天记录会导出带思维链的内容**。如果你的聊天记录很长，请先手动向上滚动，直到加载出需要导出的所有楼层。
    * 面板上方会显示“当前已加载 XX 条”。
5.  **导出**：点击对应的格式按钮（EPUB/HTML/Word...）即可下载电子书。
6.  **可能遇到的问题**
    - 部分浏览器会自动给epub格式加zip后缀，删除zip即可
    - doc文件在手机wps打开可能会缺失渲染，在电脑端是正常的
